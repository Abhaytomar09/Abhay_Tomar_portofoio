# ⚡ Abhay Tomar — Personal Portfolio


> **Live Site:** [https://abhaytomar09.github.io/Abhay_Tomar_portofoio/](https://abhay-tomar-portofoio.vercel.app/)

A futuristic, fully animated **cyberpunk-themed personal portfolio** built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, just pure web tech with premium aesthetics.

---

## 🚀 Features

| Category             | What's Inside                                                                   |
| -------------------- | ------------------------------------------------------------------------------- |
| 🎨 **Design**        | Cyberpunk dark theme · Neon cyan/purple/pink palette · Glassmorphism cards      |
| ✨ **Animations**    | Terminal typing loader · Glitch name effect · Typewriter roles · Scroll reveal  |
| 🖱️ **Interactivity** | Custom cursor + 6-particle neon trail · 3D card tilt · Skill tag ripple         |
| 📊 **Sections**      | Hero · About · Skills · Projects (3) · Education · Achievements · Contact       |
| 📱 **Mobile**        | Fully responsive · Hamburger menu · Mobile-first breakpoints                    |
| ⚡ **Performance**   | Scroll progress bar · Lazy-loaded photo · sessionStorage loader skip on refresh |
| 📬 **Contact**       | Working Formspree AJAX form · No page redirect · Inline success message         |
| 🌐 **PWA**           | Installable as mobile app via `manifest.json`                                   |
| 🔗 **Social**        | OG image for LinkedIn/WhatsApp previews · Live GitHub contribution graph        |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure, OG/PWA meta tags
- **CSS3** — Custom properties, glassmorphism, keyframe animations, CSS Grid/Flexbox
- **Vanilla JavaScript** — Canvas particles, IntersectionObserver, Fetch API, sessionStorage
- **Formspree** — Backend-free contact form
- **[ghchart.rshah.org](https://ghchart.rshah.org)** — Live GitHub contribution graph embed
